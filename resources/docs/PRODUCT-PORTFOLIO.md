## 1. Privacy Shield

**One-line pitch:** Drop-in API proxy that strips PII from LLM requests and restores it in responses — use any AI while keeping sensitive data local.

**Target customer:**
- Enterprises with compliance requirements (HIPAA, GDPR)
- Developers building AI apps with sensitive user data
- Organizations that can't send customer data to cloud APIs

**Price point:**
- Open source (self-hosted): Free
- Managed cloud: $500-1000/month
- Enterprise support: Custom

**Deployment complexity:** ★★☆☆☆ (Docker one-liner)

**Competitive advantage:**
- Actually works (18 custom entity recognizers including API keys, cloud creds, internal IPs)
- <10ms latency overhead
- OpenAI-compatible API — drop-in replacement
- Kill switch for instant bypass
- No cloud required

**Status:** Ready for Product Hunt
